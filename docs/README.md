# AI Agent Swarm Codex-only Docs

用户通常只需要三个入口：

1. `INSTALL_PROMPT.md`：安装后验证插件。
2. `START_PROMPT.md`：日常开发、新项目启动、已有项目接手的统一入口。
3. `RELEASE_PROMPT.md`：发布包和 GitHub Release 同步检查。

支持文档：

- `CUSTOM_AGENTS.md`：Codex Custom Agents 如何安装、使用和关闭。
- `ENGINEERING_GATE.md`：非简单任务的设计、计划、审查、测试和验收闸门。
- `OFFICIAL_DOCS_GATE.md`：何时必须查官方文档并记录证据。
- `RAG.md`：本地项目记忆库规则。
- `SUBAGENT_WORKFLOW.md`：可见子智能体角色流程。
- `ROADMAP.md`：后续计划。
