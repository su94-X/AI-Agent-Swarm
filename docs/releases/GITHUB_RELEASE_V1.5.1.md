# AI Agent Swarm V1.5.1 Release Notes

AI Agent Swarm `V1.5.1` is a documentation refresh release for the GitHub landing page.

- Reworked `README.md` into a clearer public project homepage.
- Added a visible comparison between the full workflow and Lite branch.
- Moved Custom Agents usage guidance closer to the top of the README.
- Clarified that `.codex/agents/*.toml` templates must be copied into the active project or user-level agent directory to be loaded by Codex.

- Clarified RAG, safety boundaries, package-release checks, and the three prompt entrypoints.
